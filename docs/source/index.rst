Kable Documentation
===================

A lightweight terminal IDE built with Textual.

Kable combines a file explorer, text editor, and Git-aware project navigation
into a keyboard-driven interface designed for terminal workflows.

Features
--------

* Tree-based file explorer
* Built-in text editor
* Syntax highlighting
* Git status integration
* Keyboard-driven workflow
* Lightweight and terminal-native
